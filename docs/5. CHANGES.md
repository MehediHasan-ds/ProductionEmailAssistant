# Changes from the Plans

This file records where the build diverges from the earlier docs. I am not editing those, since they capture my thinking at the time. Where this file and the others disagree on providers, this file wins.

## What changed

The original plans named three providers, OpenAI, DeepSeek, and Gemini. DeepSeek is not free, so I dropped it, and I also dropped direct OpenAI. The final set is two models:

- gpt oss 120B, accessed through OpenRouter
- gemini 2.5 flash lite, accessed through Google AI Studio

The exact identifiers are:

```
OpenRouter: base url https://openrouter.ai/api/v1, model openai/gpt-oss-120b
Google AI Studio: base url https://generativelanguage.googleapis.com/v1beta/openai/, model gemini-2.5-flash-lite
```

## What this means

Almost nothing structurally. Both providers speak the OpenAI chat format, so the provider agnostic client from the architecture still works. I only swap the base URLs and model ids in configuration. The local Jina embeddings, the hybrid metrics, the self refinement loop, the twelve scenario evaluation set, and the non functional targets are all unchanged.

## Environment

The two keys live in my gitignored .env file as OPENROUTER_API_KEY and GEMINI_API_KEY, so they never enter the repo.
